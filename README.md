# MERN Stack Social Media Site [![license](https://img.shields.io/badge/license-MIT-blue)](https://shields.io)

This is a social media web app used for Developers to connect to one another. 

## Tech Used
* MongoDB
* Express
* React
* Node.JS
* Redux
* Packages
    * bcryptjs
    * config
    * express-validator
    * Gravatar
    * json-web-token
    * mongoose
    * request
